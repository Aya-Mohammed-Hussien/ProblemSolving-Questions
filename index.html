16-2-2025 Problem: Array Chunking

Write a JavaScript function that splits an array into subarrays (chunks) of a specified size. The function should take two arguments:

1- The array to be chunked. 2-The size of each chunk. Requirements: 1- If the array is empty, return an empty array.. 2- If the chunk size is less than 1, return an empty array.
